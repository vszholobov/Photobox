vkbot module¶

vkbot.bot_typing(user)¶

Функция, имитирующая печатание сообщение бота при общении с пользователем.

Параметры

user – кортеж из индентификатора и случайного числа.

vkbot.random_search(user, message)¶

Функция делает запрос на сервер и затем отправляет случайную фотографию пользователю.

Параметры

  • user – кортеж из индентификатора и случайного числа в виде кортежа.

  • message – сообщение для пользователя.

vkbot.send_photo(user, route, message)¶

Функция, обеспечивающая боту отправление фотографий пользователю.

Параметры

  • user – кортеж из индентификатора и случайного числа в виде кортежа.

  • message – сообщение для пользователя.

  • route – директория фотографии.

vkbot.tag_search(user, message, emoji, keyboard)¶

Функция по тэгам ищет фотографии: делает запрос на сервер и возвражает директории фотографий.

Параметры

  • user – кортеж из индентификатора и случайного числа в виде кортежа.

  • message – сообщение для пользователя.

  • emoji – список эмодзи.

  • keyboard – клавиатура в виде словаря.

Результат

список директорий фотографий.

vkbot.write_msg(user, message, emoji, keyboard)¶

Функция отправляет сообщение пользователя через бота.

Параметры

  • user – кортеж из индентификатора и случайного числа.

  • message – сообщение для пользователя.

  • emoji – список эмодзи.

  • keyboard – клавиатура в виде словаря.
