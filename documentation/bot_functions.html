bot_functions.analyze_command(coincidence, text, odds)¶

Функция провряет сообщение, полученное из text на наличие команды, полученной из coincidence, c точностью odds.

Параметры

  • coincidence – команда для сравнения с text.

  • text – строка, которую отправил пользователь боту.

  • odds – параметр, влияющий на точность выбора подходящей команды (чем меньше, тем точнее).

Результат

«Nope», если функция не распознала команду в text по длине, иначе подходящая команда.

bot_functions.analyze_text(text, commands)¶

Функция получает на вход две переменные, по которым определяются дальнейшие действия бота. Обработка текста полученного из переменной text происходит с помощью вычисления редакционного расстояния (расстояния Левенштейна), после выполнения которого выбирается наиболее подходящая команда из переменной commands.

Параметры

  • text – строка, которую отправил пользователь боту.

  • commands – команды, которые бот может распознать в text.

Результат

«Nope», если функция не распознала команду в text, иначе подходящая команда.

bot_functions.create_photo_matrix(photos, user_id)¶

Функция создает матрицу из фотографий, находящихся в директориях, полученных из photos, и сохраняет её в директории ботов c именем, полученным из user_id. Как работает: каждой ячейке матрицы функция присваивает номер, который потом нужен для выбора фотографии пользователем, этот выбор зависит от высоты фотографии. Затем составляются четыре колонки и в колонку, с наименьшей высотой добавляется фотография в матрицу.

Параметры

  • photos – список директорий фотографий.

  • user_id – Индентификатор пользователя.
