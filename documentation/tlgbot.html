tlgbot module¶

tlgbot.create_keyboard(min_border, max_border)¶

Функция создает клавиатуру, которая выведется пользователю.

Параметры

  • min_border – минимальное значение диапазона клавиатуры.

  • max_border – максимальное значение диапазона клавиатуры.

Результат

квавиатура в виде списка списков.

tlgbot.message_handler(bot: telegram.bot.Bot, update: telegram.update.Update)¶

Функция обработки сообщений, приходящих от пользователя.

Параметры

  • bot – объект bot.

  • update – объект update.

tlgbot.random_search(bot: telegram.bot.Bot, update: telegram.update.Update)¶

Функция делает запрос на сервер и отправляет случайную фотографию пользователю.

Параметры

  • bot – объект бота.

  • update – объект update.

tlgbot.send_photo(bot: telegram.bot.Bot, update: telegram.update.Update, first_keyboard)¶

Функция заставляет бота отправить фотографию пользователю.

Параметры

  • bot – объект бота.

  • update – объект update.

  • first_keyboard – стартовая клавиатура.

tlgbot.tag_search(message, chat_id)¶

Функция делает запрос на сервер и возвращает список директорий подходящих к тэгам фотографий.

Параметры

  • message – сообщение пользователя.

  • chat_id – индентификатор пользователя.

Результат

список директорий к фотографиям, если есть хотя бы одна фотография, иначе 0.
