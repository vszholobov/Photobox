server_functions.join_templates(tags, description)¶

Функция соединяет тэги и описание в один список.

Параметры

  • description – описание фотографии.

Результат

список из тэгов и описания.

server_functions.save_picture(form_picture, picture_fn, path='static/profile_pictures')¶

Сохраняет фотографии под случайным 16-значным именем в static/profile_pictures.

Параметры

  • form_picture – объект фотографии

  • picture_fn – имя фотографии

  • path – директория для сохранения фотографий.

Результат

возвращает отношение высоты к ширине фотографии.

server_functions.size_analise(image)¶

Функция обрезает фотографию, делая из нее квадрат со стороной равной длине меньшей стороны изначальной фотографии, при этом центрируя новую фотографию относительно исходной.

Параметры

image – объект фотографии.

Результат

измененный обьект переданной фотографии.

server_functions.size_t_analise(image)¶

Функция определяет размер фотографии, исходя из соотношения её сторон.

Параметры

image – объект фотографии.

Результат

размер фотографии в виде кортежа.

server_functions.sort_pictures_by_tag(list_of_pictures, list_of_tags)¶

Сортирует фотографии по тегам, выбраным пользователем.

Параметры

  • list_of_pictures – список объектов фотографий.

  • list_of_tags – список тэгов.

Результат

возвращает кортеж, состоящий из списка директорий и списка индентификаторов пользователей.
