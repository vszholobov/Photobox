errors module¶

errors.internal_error(error)¶

Функция, работающая с ошибками связанными с сервером.

Параметры

error – код ошибки.

Результат

страница с кодом ошибки 500.

errors.not_found_error(error)¶

Функция, отвечающая за работу с несуществующими страницами.

Параметры

error – код ошибки.

Результат

страница с кодом ошибки 404.
